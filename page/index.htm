SNAAS

Strange Names As A Service

Inspired by Fuck off as a service and the AngularJS naming scheme (link), this is a service that builds strange name combinations like "impure skiing" or "upright zephyr" so you don't have to.

Choose between arbitrary adjective-noun combinations, random alliterations or alliterations starting with your desired literal.

Multilingual

Multi language support, just provide your desired language in the Accepted-Language Header:

Accept-Language: en

Currently supported languages: English

Restful API

SNAAS offers a restful API to generate your own strange names just as you desire.

View details »

API

            
/api/name

            Returns a strange name in a arbitrary adjective-noun combination.
            
Try »


            
/api/alliteration/:letter?

            Returns a alliteration. The parameter letter is optional and specifies the letter the alliteration begins with.
